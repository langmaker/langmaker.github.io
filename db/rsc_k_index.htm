Khoisan Linguistics - Some information on several Khoisan languages of southern Africa, famous for their clicks.

Kura - Kura (Nepali for language) is a multi-user open-source linguistic database especially geared towards language description... Kura is an application for linguists working with fieldwork or manuscript data. It supports the entry, analysis and presentation of linguistic data, be it recordings or manuscripts. It is a multi-user, multi-language, multi-project application, meaning that several linguists can simultaneously work with all data, and that the data is not restricted to one language. (Not available for Windows.)

Kurdish Language Studies - Kurdish resources from the University of Arizona. Includes pronunciation guide, spell-checker, and dictionary.

KZod - A program to create your own dictionaries and add new records to existing ones. It has the ability to create chronological word lists. Supports DICT, MOVA and SLOWO formats.
