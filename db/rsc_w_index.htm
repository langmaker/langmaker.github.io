Weird Words - Recently, I read on Langmaker that John Harper was looking for funny words to use in a language, and I thought this might help him and like-minded people.

West African Languages -

Friends of Guinea is an organization of former Peace Corps volunteers serving in Guinea, West Africa. The Guinean Languages section of the FOG website has hard-to-find material on Susu, Pulaar (Fula) and Malinke languages, as well as Wolof from Senegal. Pulaar especially would be of interest to conlangers due to its amazing complexity including an honorific vocabulary entirely different from the standard vocabulary.

There is a link to a site for a sort of Pan-Mande language called N'Ko that uses common features of the related languages of Bambara, Malinke, Mande, Mandingo and others to create a lingua franca. Follow the link for some cool N'Ko alphabet info with font download.

There are also other links pertaining to African languages.

When Do People Learn Languages? - An excellent overview of why and when people learn specific languages, with some advice on how to learn a language.

Word2Word Free Online Language Courses - This site has free language courses from Abenaki, to English, to Xhosa. Most of the courses are excellent, and many are beginner courses, which are great to get a bit of the feel of a language but not to become fluent.

Word Creator - A simple word creator written in Java. You will need the Java virtual machine, version 1.4 or later.

wordgen - Wordgen is a Perl script for automatically generating a random character sequence based on a rule set. Its main purpose is automatic word generation for conlanging, but it may have other uses.

WordGen - This is version 1.8 of a powerful software application for word creation. It uses syllable structure and includes a dictionary. It is now available in English. Contact bvilches@gmail.com with comments.

Word Generator Scripts - Some word generator scripts that I've made, which allow you to define your own phonology.

WordGumbo - A site containing word lists of several languages, including some that are real hard to find via search engine. Good if you are looking for a word list for deriving a conlang.

WordSynth.pl - "WordSynth is a perl script that randomly generates words based on a language rules file. It was inspired by GDW's Traveller language generation system, but includes many enhancements. While it still takes some work to create a good language rules file, WordSynth makes it easy to perform tweaks and tests until usable results are reliably got." - Ken St-Cyr

World Creator - The second generation of my PCALD software. Presently it is not backward compatible with PCALD, but I am working on that. Since the whole thing is stored in a JAR (Java ARchive), it is much easier to run. You just need to make sure you have Java on your machine and then you can hopefully just double-click on the WorldCreator JAR in the directory WorldCreator, to run it.
