Camion Code - A "universal code for the common language". Created in 1999 by Jean Camion, this system of writing claims to be able to encompass all phonemes.

Cerulean

Choton Handwriting - A handwriting script for quickly and efficiently writing Choton (and other languages). You can see the Babel text in Choton handwriting.

Complanian

Corbacian - This is a complex syllabary, whose characters are built from three sets of parts. (The "gat x,y" in the Script Image are instructions for how to combine these parts, in order to get correctly formed Corbacian characters. If you look at the yellow boxes of the vowels, you should be able to understand how the system works.)

Corrie - A script for a new language I'm working on. It is written from right to left without spaces. Vowel diacritics are used along with a special mark to divide words. The language technically has no verb forms, but rather various forms of nouns and pronouns inflect action or a state of being.

Cthulhu - Cthulhu is an alphabet inspired by the transdimensional monsters of the American science-fiction writer H.P. Lovecraft and intended for use in a forthcoming sequel to David M. Mitchell's Lovecraft tribute The Starry Wisdom (Creation Books). Voiced plosives and fricatives are distinguished from unvoiced by use of a dot, but all other consonants, including 'c', have a unique form. Vowels are based on star patterns. There is no distinction between upper- and lowercase.

CyrDaiLue - This script is derived from Cyrillic (Russian), but it uses the Thai Lue vowel system (which ultimately is of Indic origin).
