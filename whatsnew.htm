What's Old at LangMaker.com?

If you want to receive an e-mailed announcement whenever this site is updated, just click here.


Click to subscribe to What's New At LangMaker.com

Note: This page has been deprecated in favor of showing what's new on the home page itself.

Date Addition
7/3/99
  • Added a proposed Folkspraak grammar by Dan Dawes.
  • Added a Folkspraak discussion list.
  • Kardasi (Schrager, Esther) - fictional diachronic language - 1994.
  • Mango (Gruscha, Natalia) - fictional language - 1997.
  • Updated the Interlingua link and added notes based on commentary from Chris Burd.
  • Updated the Volapük link and the Dutton World Speedwords link.
6/25/99
  • Added Babel Text pages for Dha-Patu, Mango, NGL and Nuirn.
  • Updated the Folkspraak page, which still interests people, despite being dormant for three years. We need someone to go through the Folkspraak notes and make some decisions and breathe some life back into this project.
6/23/99
  • Breathanach (Eddy, Geoff) - fictional diachronic language - 1998.
  • Denden (Rempt, Boudewijn) - fictional language - 1984.
  • Eastern (Taylor, Nik) - fictional diachronic language - 1999.
  • Gladilatian (Himes, Dennis Paul) - fictional language - 1997.
  • Ilianore (Smith, Jeff) - fictional diachronic language - 1996 (entry updated).
  • Lakal/Saradic (PJS) - fictional language - 1994.
  • Low Orkish (Zimmermann, Merle D.) - fictional language - 1999.
  • Sakatda Ka Kadomo (Garcia, Barry) - personal language - 1999.
  • Speedtalk (Heinlein, Robert A.) - international auxiliary language, shorthand - circa 1940.
  • Üqoi (Bracken, Bruce V.) - fictional language - 1996.
  • Vulcan (Roddenberry, Gene) - fictional language - 1990s.
  • Watakassí (Taylor, Nik) - fictional diachronic language - 1998.
  • Added Babel Text links for Breathanach, Ok and Surfarian.
5/16/99
  • We have been providing custom-consulting services to writers and businesses that need assistance creating model languages.
  • Added Europanto (Marani, Diego) - international auxiliary language - 1996.
  • Added Lin (Skrintha) - fictional language - 1996.
  • Added Thosk (Easton, Dean) - fictional diachronic language - 1992.
  • Added Vosh (Myers, Daniel) - professional fictional language - 1999.
  • Added Bahasan Babel Text by Leo Moser.
  • Added Portuguese Babel Text translated by Roberto Machado de Campos.
  • At the request of Tolkien linguist Helge K. Fauskanger, some of the Tolkien languages listed on the Model Language Index were reclassified, a few links were repaired and more detail was added to Westron.
5/2/99
  • Added Andorian (Hill, Spence) - fictional language - 1990.
  • Added Hani (Cherryh, C.J., documented by Spence Hill) - professional fictional language - 1998.
  • Added Idrani (Pehrson, T. Mitchell) - personal language - 1998.
4/29/99
  • Added Leksventin translation of the Babel Text.
  • Added missing </table> tag to home page, since its absence had caused the last version of the page to appear blank in Netscape Navigator. Oops!
4/27/99
  • For the most detailed family of model languages on the web, you simply have to check out Mark Rosenfelder's Almean languages: Ismain, Kebreni, Verdurian, and Wede:i.
  • Added Ptung, it-Penginijtung (Lawless, Anthony; Gallagher, Evan; Havlis, Jan) - personal language - 1996.
4/23/99
  • Added Syldavian (Hergé, restored by Mark Rosenfelder) - fictional language - restored 1997.
  • Added Xhamagas - (Fredin, Johan) - fictional language - 1998.
4/21/99
  • Implemented some of B.Philip Jonsson's suggestions for improving the Submission to Model Language Index form. Now is as good a time as any to submit your own favorite language links.
4/20/99
  • Fixed link to Blaaninain (Lord Blaa) - fictional language - 1998.
4/19/99
  • Added write-up of Furbish, the language of the Furby, to the site.
  • Added Arkian (Havlis, Jan) - fictional diachronic language - 1995 to index and Babel Text.
  • Added Vendi Babel Text translation.
  • Fixed those Ardalambian links in the index that were relative instead of absolute.
4/14/99
  • Added every Tolkien-related language documented on Ardalambion: marked with in the index.
  • Added link to Tolkien, J.R.R. - The Silmarillion, The Hobbit, The Lord of the Rings.
  • Added pala-kalloejna (Reid Palmer, Christopher Reid) - personal language - 1997.
  • Added Rick Harrison's a system of classifying constructed languages to the classifications page.
  • Moved a lot of dead links here. Let me know about other dead links on the Model Language Index page.
4/13/99
  • Added survey to home page; please take part!
  • Updated the Babel Text to include translations originally uploaded at Nik Taylor's mirror site, including the constructed languages Almaqerin, Draseléq, Dzalze, Géarthuns, Keleñ and Rokbeigalmki and the natural languages Danish and Polish.
  • Added Futurama (Groening, Matt, et al) - professional fictional languages - 1998.
  • Added Kobaian (Vander, Christian) - fictional language.
  • For good ideas on how to summarize your language's grammar, check out this Grammar Sketch Outline, originally intended for natural languages.
4/07/99
  • Added Adelic (MacLagan, Scott) - fictional diachronic language - 1993.
  • Added Danovën/Arovën (Shinavier, Joshua) - logical language - circa 1994.
  • Added Kankonian (Landau, James E. F.) - fictional language - 1997.
  • Added Picture Language (Flint, Wally) - international auxiliary language - 1990.
  • Added Vendi (Ashton, Chris) - Personal language - 1993.
4/03/99
  • Uploaded the complete, 850 word list for Basic English. Useful as the basis for starting your own lexicon.
  • Re-uploaded home page; the bottom link to the Scattered Tongues web ring wasn't showing up.
  • Added the What's New At LangMaker.com mailing list for people who want to be e-mailed each time the site is updated.
  • Added the LangMaker2 mailing list for general discussions of language making.
  • For the sheer joy of it, you should check out how Tolkien's Tengwar alphabet can provide "a romantic orthography for Lojban". (Added this to the Lojban entry of the Model Language Index.)
  • Added Prototype Worlds (Raymond, Eric; editor) - superset language - 1996 to language index.
  • Added Finnish translation of the Babel Text, contributed by Niilo Paasivirta.
  • Added two links to D'ni, the language of the computer game Riven (sequel to Myst).
  • Uploaded a Submission To Model Language Index form so that new entries will be more complete.
2/15/99
  • Added Deviasew to the Model Language Index and added Deviasew translation of the Babel Text, submitted by Jonathan Washington North.
2/02/99
  • Added Delason to the Model Language Index, submitted by Nizar Habash.
  • Added Spanish version of the Babel Text, submitted by Carlos D. Montenegro.
  • Added Merdian version of the Babel Text, submitted by David Dynes.
1/15/99
  • Reordered FAQ on home page with most commonly asked questions listed first (except for initial question), based on analysis of which pages were visited most.
  • Inspired by the popularity of the Top 10 Model Languages list, added LangMaker's Top 10 Model Languages, listing the most requested model language pages from LangMaker.com in December.
1/14/99
  • Released version 1.10 of LangMaker fixing error on startup.
1/12/99
  • Added Blaaninain to Model Language Index and to Babel Text.
  • Joined Scattered Tongues web ring.
1/11/99
  • Added Model Language Index, Author-Oriented page.
1/8/99
  • Add Kriegsprach to Model Language Index.
12/16/98
  • Released version 1.09 of LangMaker and made it open source.
