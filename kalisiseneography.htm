Neography   Advanced
Kali-sise is typically written with the Latin alphabet.  However, while the language has been designed to be easy to pronounce, Mattias Persson and I thought it would be interesting to create a syllabary for the language to give it some visual appeal.  For inspiration, Mattias looked to the symbols of the ancient Naxi Geba script from China.

Mattias has also written Kali-sise in the Watterian abugida.
